Junior Java Developer

Home Positions Junior Java Developer

Java Developer is a specialist whose main task is to create complex corporate systems for data processing. These systems include desktop applications, web services, cloud storage, web applications, as well as complex banking systems.

Java programmers are in demand in those areas where they use projects with multi-user data processing and high requirements to the level of security. A Java programmer usually develops the architecture of the application, the business logic of the application, maintains and maintains the project, and writes technical documentation.

Who is Java developer?

It is a developer who creates various software using the Java programming language.

Main activities:

Using the Java programming language and certain assistive technologies, he creates:
- web applications (server side);
- computer games;
- desktop applications;
- mobile applications for Android;
- cloud data storage;
- software for banking and corporate systems.
- support and updating of existing applications.

Java developer requirements:

- Deep knowledge of the Java programming language;
- Strong knowledge of OOP and design patterns;
- Strong knowledge of algorithms and data structures;
- Knowledge of databases (MySQL or PostgreSQL);
- Knowledge of JDBC (API for interacting Java with databases) and Hibernate;
- Knowledge of application testing systems (TDD);
- Knowledge of XML markup language and JSON web technology;
- Basic knowledge of the Spring framework;
- English at a level not lower than Intermediate.

Java Database Connection: JDBC and MySQL

In this course, you learn how to connect to a MySQL database using Java JDBC. The course starts with an overview of the JDBC API. Then you learn how to set up your development environment with the appropriate MySQL database drivers. Next the course shows you how to submit a SQL query and process the result set. Also, you learn how to perform SQL insert, updates and deletes. The course moves on to advanced topics such as Prepared Statements to handle SQL parameters. You also learn how to call stored procedures using various parameter types (IN, INOUT, OUT and ResultSet). Next, the course shows you how to process large data types such as BLOBs and CLOBs. Finally, the course wraps up with a section on reading database connection information from a configuration file.
